## Context

Create the first production-quality template: the "explainer" template for standard explainer video content. This template should make title cards, section transitions, bullet reveals, and metric displays look polished and professional.

The template renders:
- Title cards: large centered text with theme colors, subtle background treatment
- Section headings: clean transitions between sections
- Show text: readable body text with appropriate spacing
- Metrics: prominent value display with directional indicators (up/down/neutral)
- Steps: staggered list items with clear visual hierarchy

All styling uses CSS custom properties from the theme system (var(--moron-*)).

Files: packages/ui/src/templates/ExplainerTemplate.tsx (new)

## Acceptance Criteria

- Renders all 5 element types (title, section, show, metric, steps)
- Professional visual quality using theme CSS properties
- Responsive to frame dimensions (works at 1080p and 4K)
- Z-ordering and opacity/transform from FrameState respected
- No hardcoded colors â€” all values from theme custom properties
- Renders correctly in headless Chromium
